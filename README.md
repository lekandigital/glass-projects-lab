# glass-projects-lab

Live gallery: https://glass-projects-lab.vercel.app
GitHub repository: https://github.com/lekandigital/glass-projects-lab

This repo is a catalog of glass / liquid-glass experiments with a mix of:

- production Vercel deployments
- local relative demos served directly from the catalog
- a few static `/dist/` examples kept intentionally local

## Deployment notes

- The repo root is the catalog site and lives at `https://glass-projects-lab.vercel.app`.
- Relative links stay relative when the catalog is served from the repo root.
- App-style folders should be deployed from the actual app folder, not the collection root.
- Monorepos like `liquid-dom-master` need the real showcase app path, not the repository root package graph.
- The Vercel aliases for some long project names are shortened automatically. That is expected.
